# chull

convex hull from points

```
pip install chull
```

- [Documentation](https://chull.readthedocs.io/)
- [chull on Github](https://github.com/szabolcsdombi/chull/)
- [chull on PyPI](https://pypi.org/project/chull/)

## Usage

```py
points = np.array([
    ...
])

vertices, normals = chull.make_hull(points)
```
